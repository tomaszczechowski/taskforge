# @taskforge-ai/jira-client

Jira REST API v3 client for issue fetching, commenting, and workflow transitions used by the Taskforge agent.

## Capabilities

- Fetch issues assigned to a given user
- Post and read comments (used for the approval gate)
- Transition issue status (e.g. "In Progress" → "In Review")

## Part of the Taskforge monorepo

See the [main README](https://github.com/tomaszczechowski/taskforge) for full documentation, configuration reference, and architecture overview.
