# @taskforge-ai/github-client

Thin GitHub API client for branch and pull request management used by the Taskforge agent.

## Capabilities

- Create and delete branches
- Open pull requests
- Fetch PR status and review state

## Part of the Taskforge monorepo

See the [main README](https://github.com/tomaszczechowski/taskforge) for full documentation, configuration reference, and architecture overview.
