# @taskforge-ai/shared

Shared types, Atlassian Document Format (ADF) utilities, and the logger used across the Taskforge monorepo.

## Exports

| Export | Description |
|---|---|
| Types | Common interfaces and enums (`TicketState`, config shapes, etc.) |
| ADF helpers | Convert Jira ADF body to plain text |
| `logger` | Structured console logger used by all packages |

## Part of the Taskforge monorepo

See the [main README](https://github.com/tomaszczechowski/taskforge) for full documentation, configuration reference, and architecture overview.
